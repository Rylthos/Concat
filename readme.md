# Concat
Stack based programming language

## Goals
- Compile to assembly
- Implement a type system to ensure safety
- Bootstrap the compiler

## Language Features
- Basic arithmetic and control flow

## Examples
- fibonacci
- fizzbuzz

## Language syntax
Arithmetic operations
```text
+ - * / %
```

Stack Operations
```text
rot3 dup drop over swap cast print
```

Boolean Operations
```text
< > <= >= = != && || !
```

Loops
```text
if else while
```
